
# Student Portal (HTML)

## 📌 Description
This project is a Student Portal interface developed using HTML. It provides a structured layout for user authentication, including login and signup pages. The system is designed to simulate a basic student management interface, focusing on clean design and user-friendly interaction.

## 🚀 Features
- User Login Form
- User Signup (Registration) Form
- Input validation structure (HTML-based)
- Organized and responsive layout (basic)

## 🛠 Technologies Used
- HTML
